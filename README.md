My first React frontend project built using React and Vite.
This project is a simple and responsive profile card UI with interactive buttons, hover effects, and modern styling.

**Features**
- Responsive profile card
- Gradient UI design
- Hover Effect
- Functional GItHub hutton
- Functional Contact button
- Built with React + Vite

**Technologies Used**
- react
- Vite
- CSS5
- JavaScript

**What I Learned**
Through this project, I learned:
- React Project setup
- Components and JSX
- styling in react
- Flexbox alignment
- Adding functional buttons and links
- Running a development server

**Inspiration**
This project was originally inspired by a BroCode YouTube tutorial, but I customized the design and styling and functionality by adding my own ideas and changes while learning React.
